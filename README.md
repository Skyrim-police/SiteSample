# Otech - Професійна розробка програмного забезпечення

## Опис проекту

**Otech** — це сучасна компанія з розробки програмного забезпечення, що спеціалізується на створенні високопродуктивних додатків з використанням передових технологій. Ми фокусуємося на розробці комплексних рішень для бізнесу, включаючи онлайн-платформи та банківське програмне забезпечення.

Цей веб-сайт створено для представлення нашої компанії, технологічного стеку та проектів, над якими ми працюємо.

## Особливості сайту

- Сучасний та адаптивний дизайн
- Повна підтримка мобільних пристроїв
- Плавні анімації при прокручуванні (AOS)
- Швидке завантаження
- Повністю українською мовою
- Доступний інтерфейс
